# 物理安全
### 概述
物理安全技术包括:
防盗、防火、防静电、防雷击、防信息泄漏、物理隔离；
基于物理环境的容灾技术和物理隔离技术也属于物理安全技术范畴。

物理安全是信息安全的必要前提。
如果不能保证信息系统的物理安全，其他一切安全内容均没有意义。
